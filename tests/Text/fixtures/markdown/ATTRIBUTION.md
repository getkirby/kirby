# Markdown input fixtures attribution & licenses

The markdown **inputs** under [`inputs/`](inputs/) are drawn from third-party sources. Only the inputs are reused; the expected HTML in [`snapshots/`](snapshots/) is generated by Kirby's own pipeline and is not derived from any third party.

The sources and their licenses:

| Directory                | Source                                                                               | License                  |
| ------------------------ | ------------------------------------------------------------------------------------ | ------------------------ |
| `inputs/commonmark/`     | [CommonMark spec](https://github.com/commonmark/commonmark-spec) `spec.txt` examples | CC BY-SA 4.0 (see below) |
| `inputs/parsedown/`      | [Parsedown](https://github.com/erusev/parsedown) 1.8.0 `test/data`                   | MIT (see below)          |
| `inputs/parsedownextra/` | [ParsedownExtra](https://github.com/erusev/parsedown-extra) 0.9.0 `test/data`        | MIT (see below)          |

## CommonMark spec — CC BY-SA 4.0

The files in `inputs/commonmark/` are the _input_ halves of the example blocks in the CommonMark spec (`spec.txt`).

> The CommonMark spec (spec.txt) is Copyright (C) 2014-16 John MacFarlane,
> released under the Creative Commons CC BY-SA 4.0 license:
> <https://creativecommons.org/licenses/by-sa/4.0/>.

**Changes made:** each file contains only the markdown input of one example
(the expected-HTML half is omitted), and the spec's visible `→` tab markers have been replaced with real tab characters. The expected outputs in `snapshots/` are **not** the spec's reference HTML. They are produced by Kirby's current Parsedown-based pipeline.

Because CC BY-SA 4.0 is a ShareAlike license, these specific input files remain licensed under CC BY-SA 4.0. This applies only to the files in
`inputs/commonmark/`; it does not affect the license of the rest of Kirby.
